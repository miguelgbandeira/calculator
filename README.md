# Calculator

This is a simple calculator app created using vanilla HTML, CSS, and JavaScript. The purpose of this project was to practice the fundamentals of these technologies by implementing a basic calculator functionality.

## Demo

You can view a live demo of the Calculator [here](https://miguelgbandeira.github.io/calculator/). 

## Features

- Users can perform basic arithmetic operations such as addition, subtraction, multiplication, and division.
- The calculator displays the current input and result in real-time.
- Users can clear the input or reset the calculator.

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Use

- Clone or download the repository to your local machine.
- Open the index.html file in your web browser.
- Use the buttons provided to input numbers and perform operations.
- Click the "=" button to compute the result.
- To reset the calculator, click the "AC" button.
